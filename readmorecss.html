Apa itu CSS

CSS (Cascading Style Sheets) adalah bahasa pemrograman yang digunakan untuk mengontrol tampilan dan gaya elemen-elemen dalam sebuah dokumen HTML. CSS memisahkan presentasi (tampilan visual) dari struktur (konten) dokumen, memungkinkan pengguna untuk secara terpisah mengatur tampilan dan gaya dari elemen-elemen halaman web. Dengan menggunakan CSS, Anda dapat mengubah warna, ukuran, jenis font, jarak antar elemen, tata letak, gambar latar belakang, dan banyak lagi. CSS memungkinkan Anda untuk memberikan tampilan yang konsisten pada semua halaman dalam situs web Anda. Berikut adalah beberapa konsep penting dalam CSS: Selektor: Selektor digunakan untuk memilih elemen atau kelompok elemen tertentu yang ingin diberikan gaya. Misalnya, Anda dapat menggunakan selektor elemen (h1, p, div), selektor kelas (.kelas), atau selektor id (#id) untuk memilih elemen yang sesuai. Properti: Properti digunakan untuk mengatur gaya dan tampilan elemen yang dipilih. Misalnya, properti color digunakan untuk mengatur warna teks, font-size untuk mengatur ukuran font, dan background-color untuk mengatur warna latar belakang. Nilai: Setiap properti dalam CSS memiliki nilai yang dapat diatur untuk mengubah tampilan elemen. Misalnya, nilai red atau #FF0000 untuk properti color, dan nilai 16px atau 1.2em untuk properti font-size.Kaskade: CSS memiliki konsep kaskade, yang berarti gaya-gaya yang didefinisikan dapat ditumpuk atau diwarisi oleh elemen-elemen yang lebih spesifik atau berada dalam struktur hierarki yang lebih dalam. Selektor Pseudo-Class dan Pseudo-Element: Pseudo-class dan pseudo-element digunakan untuk mengaplikasikan gaya pada elemen berdasarkan keadaan atau posisi mereka dalam dokumen. Misalnya, :hover untuk menerapkan gaya saat kursor berada di atas elemen, atau ::before untuk menambahkan isi sebelum elemen tertentu. Media Queries: Media queries memungkinkan Anda mengatur gaya berdasarkan fitur-fitur perangkat yang digunakan untuk menampilkan halaman web, seperti lebar layar atau orientasi perangkat.
